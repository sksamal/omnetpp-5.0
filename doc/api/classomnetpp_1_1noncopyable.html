noncopyable Class Reference

#include <cobject.h>

Description

Utility class, to make it impossible to call the operator= and copy constructor of any class derived from it.

NOTE: dup() must be redefined as {copyNotSupported(); return nullptr;} in classes directly subclassing from noncopyable, but this is not needed in classes derived from them. For example, cDefaultList is noncopyable, so cModule and cSimpleModule no longer need to redefine dup().

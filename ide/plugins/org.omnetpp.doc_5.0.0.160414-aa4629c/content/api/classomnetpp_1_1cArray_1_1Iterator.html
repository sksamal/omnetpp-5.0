cArray::Iterator Class Reference

#include <carray.h>

Description

Iterates through elements in a cArray, skipping holes (slots containing nullptr).

Usage:

for (cArray::Iterator it(array); !it.end(); ++it) {
cObject *item = *it;
...
}

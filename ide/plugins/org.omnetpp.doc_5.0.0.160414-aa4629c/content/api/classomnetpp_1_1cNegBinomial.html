cNegBinomial Class Reference
Random Number Generation » Discrete Distributions

#include <crandom.h>

Description

Generates random numbers from the negative binomial distribution.

This class is an object-oriented wrapper around the negbinomial(cRNG *rng, int n, double p) function.

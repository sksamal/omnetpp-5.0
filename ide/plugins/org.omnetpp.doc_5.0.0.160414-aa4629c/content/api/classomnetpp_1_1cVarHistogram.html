cVarHistogram Class Reference
Statistical Result Collection

#include <cvarhist.h>

Description

Variable bin size histogram.

One may add cell (bin) boundaries manually, or let the object create cells with approximately equal number of observations in them.

The histogram range (rangemin, rangemax) is chosen after precollection of the initial observations has finished. It is not possible to add cell boundaries when histogram is already transformed.

enum TransformType

Controls the way histogram bins are set up at the end of the precollection period.

Enumerator
HIST_TR_NO_TRANSFORM 

no transformation; uses bin boundaries previously defined by addBinBound()/appendBinBound()

HIST_TR_AUTO_EPC_DBL 

automatically creates equi-probable cells

HIST_TR_AUTO_EPC_INT 

a variation of HIST_TR_AUTO_EPC_DBL

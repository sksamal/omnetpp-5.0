virtual std::string info ( ) const
inlineoverridevirtual

Can be redefined to produce a one-line description of object. The string appears in the graphical user interface (Tkenv) e.g. when the object is displayed in a listbox. The returned string should possibly be at most 80-100 characters long, and must not contain newline.
